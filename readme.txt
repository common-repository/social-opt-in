== Description ==

Plugin to add social services to WordPress while protecting the privacy of the visitors. Visitors have to activate each social service. No data is sent to any servers if the visitor doesn't want to. This plugin is totally hosted on your server.

== Installation ==

This section describes how to install the plugin and get it working.

BEST PRACTISE
1. Download `social-opt-in.zip`
2. Goto `Plugins` => `Install` => `Upload`
3. Select `social-opt-in.zip` and upload it
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to `Settings` => `Social Opt In` and configure the plugin.

ANOTHER POSSIBILITY
1. Download `social-opt-in.zip`
2. Unzip
3. Upload the containing directory `social-opt-in` to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to `Settings` => `Social Opt In` and configure the plugin.
